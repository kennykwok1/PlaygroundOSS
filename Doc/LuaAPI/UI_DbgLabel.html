UI_DbgLabel ¶

   <task pointer> = UI_DbgLabel( <parent pointer>, <order>, <x>, <y>,
                             <alpha>, <rgb>, "<font name>", <font size>, "<text string>",
                             <cmdID>, "<callback>")


   function <callback>(pTask, cmdID)
     --[[
         pTask タスクのポインタ
         cmdID 起動時に与えられた<cmdID>の値
     ]]
   end

引数 ¶

  • <parent pointer> 親UIのポインタ
  • <order> 表示プライオリティ
  • <x>, <y> 表示座標
  • <alpha> ラベル表示alpha値
  • <rgb> ラベル表示RGB値
  • <font name> 使用するフォント名
  • <font size> 使用するフォントサイズ
  • <text string> ラベル文字列
  • <cmdID> コールバックに与えられるID値
  • <callback> タップ時に呼び出されるコールバック名

戻り値 ¶

  • <task pointer> 生成された UI_DbgLabel タスクのポインタ

解説 ¶

位置、表示色、フォントとサイズを指定し、文字列を描画する。 文字列はボタンになっており、タップ操作でコールバックが呼び出される。

デバッグ用のコマンドボタンを簡易的に配置するために使用する。

コマンド ¶

(コマンド無し)
